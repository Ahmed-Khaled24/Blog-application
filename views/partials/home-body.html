    Get started

  • Make account from REGISTER.
  • If you made account before use LOGIN.

    Blog interface

  • ALL POSTS: this page contains all the posts created by all the users on this site ordered by date of creation, in this page you can use Add new post button to create posts.
  • MY POSTS: this page contains all the posts created by you ordered by date of creation, in this page you can delete any of your posts.
  • ACCOUNT: this page contains the data for the current logged in user.
  • LOGOUT: use it to log out.

    Important notes

  • If you forget your password make a new account, there is no reset password option.
  • You can make any password you like even if it is one character, there is no check for password strength.
  • Duplicate username or password is not allowed if you see strange error contains 'duplicate' word it is because you entered already used username or email, this feature is in development to make error message easier.
